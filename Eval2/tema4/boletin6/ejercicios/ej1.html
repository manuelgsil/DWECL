Ejercicios JS

Suma de números (array)

Resultado

1

Enunciado

En una web debe haber un formulario con un input type text. Al pulsar un botón, se deberá validar lo que hay en el input type text, el cual solo podrá contener números.

Si es válido, se mostrará en una etiqueta junto al input type text el resultado de sumar todos los números que contiene el input. Para hacer el ejercicio, crea un método que realice lo que se pide y que devuelva la suma o -1 si no se ha validado el input (es decir, que contiene algo que no sea un número).
