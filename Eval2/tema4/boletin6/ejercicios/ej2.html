Enunciado

2.- En un formulario habrá un input text. Al escribir sobre el input text y pulsar la tecla INTRO, iremos añadiendo elementos a un array y lo que hay en el input se borrará.
Cada vez que añadamos un elemento al array, se mostrará por pantalla el contenido del mismo en una lista no ordenada, de manera que el último elemento añadido aparezca el primero.
